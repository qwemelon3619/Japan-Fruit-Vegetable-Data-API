Japan Data API Reference

Endpoint-by-endpoint API reference with parameters, request examples, and sample responses.

Overview

Base URL: http://localhost:8080

Authentication: none (Nginx/network policy may restrict access by environment).

Data source: MAFF wholesale market data portal (seisen.maff.go.jp).

Data rights and usage conditions belong to the original provider. This API is an independent implementation.

date/from/to: YYYY-MM-DD from~to max 366 days code max length 32 q max length 100

System Endpoints

GET /health

Liveness check for API process.

Response Example

{
  "data": { "status": "ok" },
  "meta": {}
}
GET /ready

Readiness check including database ping.

Response Example

{
  "data": { "status": "ready" },
  "meta": {}
}
GET /metrics

Prometheus plaintext metrics.

Response Example

# HELP http_requests_total Total number of HTTP requests
# TYPE http_requests_total counter
http_requests_total{method="GET",path="/ready",status="2xx"} 12

  "error": {
    "code": "INVALID_ARGUMENT",
    "message": "date format must be YYYY-MM-DD"
  }
}
CodeMeaning
INVALID_ARGUMENTValidation error in request parameters
METHOD_NOT_ALLOWEDHTTP method is not supported for endpoint
DB_NOT_READYDatabase readiness check failed
DB_ERRORDatabase query/count execution failed
